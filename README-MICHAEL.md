# Für Michael – so machst du an deiner Seite weiter 🌿

Deine neue Website ist live: **https://wairua-spirit.de** 🎉
In diesem Ordner ist alles drin, um selbst weiterzubasteln – mit **Claude Code** als deinem Assistenten. Du musst nichts coden – du sagst auf Deutsch, was du willst.

## 1) Claude Code installieren
- Hol dir Claude Code hier: **https://claude.com/claude-code**
- Am einfachsten für den Anfang: die **Desktop-App** (Mac oder Windows).
- Du brauchst dafür ein Claude-Konto (Anmeldung läuft beim ersten Start).

## 2) Diesen Ordner öffnen
- Entpacke das ZIP an einen Ort deiner Wahl (z. B. in „Dokumente").
- Öffne den Ordner **`wairua-spirit`** in Claude Code.
- Claude Code liest automatisch die Datei `CLAUDE.md` und versteht dann sofort, wie deine Seite aufgebaut ist.

## 3) Einfach loslegen
Schreib Claude Code, was du möchtest – z. B.:
- *„Zeig mir eine Vorschau meiner Seite."*
- *„Ändere den Text in der Über-mich-Sektion zu: …"*
- *„Setz mein Impressum und meine Datenschutzerklärung ein."* (Text bereithalten)
- *„Mach den Kanu-Bereich etwas heller."*

Claude Code macht die Änderung und zeigt dir das Ergebnis in einer Vorschau.

## 4) Änderungen ins Netz stellen (live)
Deine Seite veröffentlicht sich automatisch über **GitHub**. Damit *deine* Änderungen live gehen, brauchst du einmalig Zugang zum Projekt:
- Erstelle dir ein **kostenloses GitHub-Konto** auf https://github.com (falls noch nicht vorhanden) und schick Lenny deinen Benutzernamen.
- Lenny gibt dir dann Zugriff aufs Projekt (oder überträgt es dir ganz – kurz miteinander klären).
- Danach sagst du Claude Code einfach **„veröffentliche meine Änderungen"** – in ~1 Minute ist es auf wairua-spirit.de live.

> Bis das eingerichtet ist, kannst du **lokal** schon alles ändern und in der Vorschau ansehen – nur das „Live-Stellen" braucht den GitHub-Zugang.

## ⚠️ Drei Dinge beachten
1. Die Datei **`CNAME` nicht löschen** – die verbindet deine Domain.
2. Bevor die Seite **bei Google** auftaucht: erst **Impressum + Datenschutz** einsetzen, dann das „noindex" entfernen (Claude Code weiß dank `CLAUDE.md`, was zu tun ist – sag einfach *„mach die Seite für Google sichtbar"*).
3. **E-Mail** (deine @wairua-spirit.de-Adresse über Google) läuft unabhängig weiter – daran ändern wir nichts.

Viel Freude damit – die Basis steht, ab jetzt ist es ganz deine Seite. 🛶
